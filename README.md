# Databehandling – IT-Högskolan 2025

Detta repo innehåller mina laborationer och analyser i kursen **Databehandling** på IT-Högskolan.  
Fokus ligger på att utforska, bearbeta och visualisera data med hjälp av bland annat **Python**, **Pandas**, **Matplotlib** och **Seaborn**.

## Innehåll
- `Lab1_College.ipynb` – Analys av amerikanska universitet (Grad.Rate, Alumni, S.F.Ratio, Expenditures)
- `College.csv` – dataset (exkluderas i .gitignore)

## Syfte
Syftet är att tillämpa grundläggande tekniker för:
- datarensning och transformation med `pandas`
- visualisering med `matplotlib` och `seaborn`
- tolkning av samband mellan olika variabler

## Verktyg
- Python 3.12  
- Jupyter Notebook  
- Pandas, Matplotlib, Seaborn
